# Perforated_Brick_Wall_GH

カーブ上に透かし積みブロックを配置するGrasshopper定義です。

## 動作環境
- Rhinoceros 8
- Grasshopper

## 使い方
1. Rhinoでカーブを作成
2. .ghファイルを開く
3. スライダーでパラメータを調整

## パラメータ
- WIDE / DEPTH / HEIGHT：ブロックサイズ
- 段数：積み段数
- 直線時のブロックの重なり：かぶり寸法
